# เกม tic tac toe
### วิธีการเล่น :
1. เมื่อเริ่มเกมผู้เล่นจะเป็นตาราง 3x3
2. โดยผู้เล่น x จะได้เริ่มก่อนและสลับกันไปเรื่อยๆ
3. สามารถเลือกช่องที่จะลงได้โดยการพิมพ์ตัวเลยแทนช่องต่างๆดังนี้

| 1 | 2 | 3 |
|---|---|---|
| 4 | 5 | 6 |
| 7 | 8 | 9 |
